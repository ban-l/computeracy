# 知识点

## 字符集

基本术语：

- 字符集为字母和符号的集合；
- 编码为某个字符集成员的内部表示；
- 校对字符指定如何比较，主要用于排序和分组。

### latin1、utf8、utf8mb4(编码)

- latin1字符集每一个字符是使用1个字节表示的（当数据库的字符集设置为latin1时，插入中文就会报错，因为一个中文至少要2个字节）
- utf8字符集每一个字符是使用1-3个字节表示的（它能够表示世界上绝大多数国家的语言，当需要插入中文选utf8字符集）
- utf8mb4字符集每一个字符是使用1-4个字节表示的

### 校验规则(校对字符)

- 校验规则使⽤utf8_ general_ ci 对select的结果不区分⼤⼩写进行筛选
- 校验规则使⽤utf8_bin 是区分大小写的
